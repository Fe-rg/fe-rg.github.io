Game Dev Tool

Ability Creation Tool

A two-part system for designing and managing in-game abilities without touching code.

Overview

For my specialization during my 2nd year at TGA I wanted to make a ability modifier system like the ones you can find in games like Path of Exile with their support gems. The modifiers are supposed to modify anything from the speed of the ability to completely changing the behavior of it, for example instead of traveling in a straight line the projectiles could instead orbit the player.

I also wanted to make it so a designer could utilize the system to create ability presets that you later could utilize in the game.

Part 1 — Editor

With the ImGui window that I'm currently using to create/change the abilities you can choose different modifiers to use and tune the settings for them. You can also reorder the modifiers to decide the order of operation to fit the feel you're going for.

When you're done setting up the ability you can then save it using json, you can then load it at a later point if you want to tweak it!

Part 2 — Runtime

How it works in-game currently is that there's base modifiers that are always applied, this could be thing like projectile speed, more projectiles or traveling behavior like orbit like I mentioned earlier.

Active modifiers however have some sort of trigger, usually collision of some kind. When the object collides it applies the current active modifier which is decided by the order set in the tool previously, it will keep applying the same active modifier on collision until said modifier runs out of "charges" after which it changes to the next modifier in line. When out of active modifiers and it collides, then the projectiles lifetime is over!
